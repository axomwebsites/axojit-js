Axojit is a open sourced and free JavaScript game engine currently under development
it supports both 3D and 2D game development. 

it is great for coders and non-coders as it has a full models library, assests (images, videos, audio) import and load, full 2D properties and drawing/painting, full 3D modeling and sculpting with advanced plugins, etc. 
and it has built-in animations and custom animating for everything in case you don't know how to code animations! 

## are you a coder?
this game engine already has built-in coding feature to code in JavaScript! 
supports Threejs for 3D and supports all JavaScript libraries! 

Do you want ui/gui for your game? (like text, buttons, frames, etc)? 
I am working on a full built-in gui system and I'm adding a feature to be able to script/code guis if you are a coder. 
even if you're not a coder, the built-in gui system has manual gui design!. 

the game engine has a built in simple customizer ai. 
note : it does NOT create new models, it only takes models and customize them depending on your request. 
for example, 
user : make a blue big house
customizer : *takes a ready house model, resize it and changes its color to blue* here is the model. *inserts model*

this game engine is completely made by Axom. 
inspired by public game engines but this one for mobile users and PC users combined
